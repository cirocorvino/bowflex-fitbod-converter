# Bowflex → Fitbod Converter

Mini web-app per convertire i setup Bowflex Xtreme SE 2 in valori prudenziali da inserire in Fitbod.

## App

Apri `index.html` oppure, quando GitHub Pages è attivo:

https://cirocorvino.github.io/bowflex-fitbod-converter/

## Come usarla

1. Scegli l'esercizio.
2. Scegli se stai usando una singola linea (`x1`) o due cavi/lati (`x2`).
3. Scegli posizione pulley e maniglia.
4. Inserisci le Power Rods usate su **una sola linea/lato**.
5. Leggi il valore finale `Fitbod` in lb e kg.
6. Copia la riga diario se vuoi salvare setup completo e conversione.

## Nota importante

I valori sono una stima conservativa, non una misurazione fisica assoluta. La mappa attuale è una v0.2 inferita. Per una v1.0 misurata servirà calibrare i setup con dinamometro o bilancia a gancio.
